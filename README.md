## 5.1 Методы
* модификатор возвращаемое значение Название метода ( параметры метода) 
{
	тело метода
}
* Переменная - значение, метод - последовательность действий
* static void ShowColor() - void это тип данных возвращаемого методом значения "ничего"
* static - модификатор метода
* ShowColor() - имя метода
* void - тип данных, означающий "ничего", это означает, что при вызове этого метода ничего не вернется
* можно открыть окно ошибок в меню вид

## 5.2 Параметры методов
* Параметры метода — значение, который метод принимает извне для использования. 
Они указываются в скобках в объявлении метода.
